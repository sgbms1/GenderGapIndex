# 市町村版ジェンダーギャップ指数 開発ガイド
― 留意点 / 都道府県版との整合 / 全国汎用化に必要なこと ―

本書は、静岡県35市区町村を対象に「4指標版」「8指標版」の市町村版ジェンダーギャップ指数(GGI)を
試作した一連の作業を踏まえ、市町村版を開発・運用する上での実務的な知見をまとめたものである。
参照する「本家」は、共同通信・上智大監修の公式**都道府県版ジェンダーギャップ指数(WEF方式)**。

---

## 1. 開発上の注意点（データ編）

### 1.1 「指定データ源に市区町村粒度があるか」を最初に確認する
- 当初 Kickoff が④育休・⑧給与の取得元に指定した**総務省「勤務条件等調査」は、全国・都道府県集計のみで
  市区町村別データが存在しなかった**。指定どおりに進めるとゴール（市区町村別比較）を満たせない。
- 機械可読の市区町村別データの主力は **内閣府「市区町村女性参画状況見える化マップ」CSV**
  （`https://wwwa.cao.go.jp/shichoson_map/data/csv/<年>/<dataname>.csv`、推進状況調査ベース）。
  - 扱う指標カテゴリは **首長／議会／公務員（管理職・係長・男性育休）／審議会／自治会／防災会議／計画** に限られ、
    **教育・経済（賃金等）の指標は無い**（→ §3.2）。
- 推進状況調査の「報告書・市区町村編」本体は **PDFのみ**。表データはマップCSVから取るのが現実的。
- 給与の男女差異は **男女共同参画局・女性活躍推進法の公表情報Excel**。ただし公表義務（常時雇用101人以上等）の
  対象自治体しか公表しないため、**小規模町村は構造的に欠損**する（本プロジェクトでは35団体中5町が欠損）。

### 1.2 コード・名寄せの落とし穴
- **JISコードの先頭ゼロ落ち**: CSVが数値化されコードの先頭0が消える（例: 青森市 02201→`2201`）。
  必ず `str.zfill(5)` で5桁化してから都道府県判定（静岡=`22`始まり）する。
- **政令指定都市の粒度差**: 統計データは市レベル（静岡市=22100、浜松市=22130）だが、
  地図ポリゴン（国土数値情報N03等）は**区に分割**（22101葵区…、22131中区…）。
  区コード→市コードの対応表で結合する（地図は親市の値で着色）。
- **Excelは団体名ベースの突合になりがち**。同名自治体（清水町／森町 など他県にも存在）を
  「(静岡県)」サフィックスで判別し、消防局・企業局・ガス局など**別法人行を厳密一致で除外**する。

### 1.3 指標値の「列の意味」を実データで検証する
- 列の見た目の推測は危険。本プロジェクトでは `shicho.csv` の列を当初「副首長**総数**」と誤解したが、
  比率列との突合（`col4 == col3/(col2+col3)*100` が35/35一致）で **「男性副首長数／女性副首長数」** と確定した。
- 出典が「総数・うち女性数」形式のときは、**男性数＝総数−女性数**として便宜計算（都道府県版と同じ流儀）。

### 1.4 欠損データの扱いを方針として固定する
- 欠損指標は**分母から除外**（入手できた指標のみで平均／分野内でウェイト再正規化）。**0埋めは構造的バイアス**を生む。
- 欠損の所在を必ず可視化（ログ出力、CSVに「使用指標数」列、地図は「データなし」をグレー表示）。

## 2. 算出ロジック上の注意点（手法編）

### 2.1 1段階目（指標ごとの男女比）
- `R = 女性値 / 男性値`（育休など一部は男性/女性）、`R̃ = min(1, R)`。**都道府県版とそのまま揃えられる**。
- **便宜指標は設計と明記が必須**:
  - ④男性育休: 女性育休取得率の市区町村別データが無い → 女性=100%を分母とみなす便宜。
  - ⑤首長・副首長: 単純な0/1二値だと首長と副市長が同価値になる。**ランク重み付き男女比（首長:副首長=2:1）**
    `R=(2×女性首長+女性副首長数)/(2×男性首長+男性副首長数)` のように役職の重みを反映できる。

### 2.2 2段階目（集約＝ウェイト付け）の最重要ポイント
- **逆標準偏差ウェイト `wⱼ=(1/sⱼ)/Σ(1/sₖ)` の罠**: 全自治体で一様に低く**分散が極端に小さい指標**
  （例: 自治会長の女性比率）に重みが集中する。本プロジェクトのフラット適用では**自治会長が約52%**を占め、
  総合スコアがその1指標に支配された。
- **分野構造（分野内を逆SD加重 → 分野間を均等平均）で単一指標支配を緩和**する。これは公式が4分野構造を採る狙いそのもの。
  分野構造化により自治会長の寄与は25%に収まり、結果が穏当になった。
- **母集団Nでウェイトも標準化も変わる**: 県内35団体で計算した相対指数か、全国1741団体基準かを**先に決める**。

## 3. 都道府県版と「揃えるべき点／揃えられない点」

### 3.1 揃えるべき・揃えやすい点
| 項目 | 内容 |
|---|---|
| 男女比の定義 | `R=女性/男性`、`min(1,R)` で1（平等）に丸める |
| 便宜計算 | 「総数−女性数＝男性数」 |
| 集約手法 | 分野内＝逆標準偏差加重、分野間＝均等平均 |
| ウェイト固定 | 基準年のデータでウェイトを固定し、経年比較を単純化する思想 |
| 透明性 | 出典・基準時点・便宜処理を明記 |

### 3.2 揃えられない・代替が必要な点
- **教育分野が最大の壁**（→ 詳細は後述）。本家の教育7指標の多くが基礎自治体になじまない:
  - 高校・大学は**市町村の管轄外**（高校＝都道府県立）。大学進学率の男女差は市区町村別に取れない。
  - 小中学校の**校長・教頭の男女比は、学校基本調査の公表が都道府県単位**で、市区町村別の性別内訳が非公表。
  - 教員給与は**県費負担**のため市町村単位で出ない。
  - 比較的可能なのは「市町村教育委員会の委員／事務局管理職の女性比率」程度だが、見える化マップには無く、
    推進状況調査PDF等からの手作業抽出が必要。
  - → 本プロジェクトでは教育の代わりに**「地域」(自治会長)** を分野に据えた（基礎自治体の意思決定層に合致）。
- **政治（首長）**: 都道府県版は「歴代知事の在職年数の男女比」。市区町村は在職年数データが揃わず、
  現職の有無＋役職重みなどの便宜にならざるを得ない。
- **経済**: 都道府県版は賃金構造基本統計・就業構造基本調査・国勢調査など多面的だが、
  市区町村別は粒度が粗いか存在しない。給与差異も公表義務閾値で欠損が出る。
- **母集団規模と小標本**: 1741団体、特に小規模自治体は分母が小さく（議員数・管理職数が一桁）、
  比率が0%や極端値になりやすい。都道府県版（47団体・規模が比較的均質）と同列に扱えない。
- **ウェイトの基準**: 都道府県は47で固定。市区町村は対象範囲（全国／県内）で標準偏差が変わるため、
  ウェイトの基準集合と基準年を**明示的に決める**必要がある。

## 4. 全国の市町村で活用できる汎用性を持たせるために必要なこと

### 4.1 データ取得の自動化・標準化
- **全国対応は容易**: 見える化マップCSVは全国収録のため、都道府県コードのフィルタを**引数化**すれば全国に展開できる
  （本プロジェクトは`22`固定 → パラメータ化が必要）。
- **取得元URL・対象年度を設定ファイル化**し、ハードコードを避ける。
- **基準時点の差を管理**: 指標ごとに年度が異なる（例: 育休=2023、他=2024、給与=令和6年度）。
  年度を揃える／差を注記する仕組みを入れる。

### 4.2 自治体マスタの整備
- ハードコードの「35件辞書」をやめ、**総務省「全国地方公共団体コード」**をマスタに採用。
- **合併・廃置分合・政令市の区**に追従できるよう、年度ごとのコード表で管理する。

### 4.3 欠損・小標本への頑健性
- 公表義務閾値による欠損（給与等）を前提に、**欠損許容ルールと最低指標数の基準**を定める。
- 小規模自治体の**比率不安定**に対し、母数（分母）の併記や信頼性の注記を行う。

### 4.4 ウェイトの基準固定
- **全国を母集団とした標準偏差でウェイトを算出し、基準年で固定**することで、自治体間・経年の比較を可能にする。
- 県内だけで再計算すると「その集合内の相対指数」になる点を明示する。

### 4.5 指標セットの共通化と「分野バランス」
- **分野ごとの指標数の偏りを是正**する（本プロジェクトは地域=1・経済=1指標）。
  逆SD加重・分野均等が意味を持つには、各分野に複数指標を揃えるのが望ましい。
- 教育を入れるなら、**全国で確実に取れる最小共通指標**（例: 市町村教育委員会委員の女性比率）に限定する。

### 4.6 透明性・再現性
- 便宜処理・欠損・ウェイト・基準年・出典を、**レポートとデータ（補助列）に必ず残す**。
- **取得 → 算出 → CSV出力 → 可視化までをスクリプトで再現可能**にする（本プロジェクトの構成を踏襲）。

### 4.7 可視化の汎用化
- 全国地図／都道府県選択、**政令市の区→市集約**、欠損の**グレー表示**、指標・分野・総合スコアの切替、
  指標の内訳・出典・年度の明示。自己完結HTML（埋め込みデータ＋CDNライブラリ）で配布性を確保。
