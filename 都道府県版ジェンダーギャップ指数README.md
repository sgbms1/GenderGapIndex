# 都道府県版ジェンダー・ギャップ指数(2026年版)

ダウンロードは[こちら](https://github.com/kyodo-official/gender-gap-index/archive/refs/heads/master.zip)

内閣府などの統計から計30指標を選出し、4分野で各都道府県の女性の地位を分析した指数。スイスのシンクタンク、世界経済フォーラム(WEF)が毎年公表するジェンダー・ギャップ指数と同様の手法で統計処理した。指数は1に近いほど平等であることを示す。

項目の選定については三浦まり・上智大教授(ジェンダー政治論)が、統計処理については竹内明香・上智大准教授(経済統計学)が監修した。

## 算出手順

本指数は、世界経済フォーラムによるジェンダーギャップ指数の算出方法に基本的に従っている。計算手順は大きく2段階となり、1段階目に30項目の男女比を算出し、2段階目に各分野ごとの加重平均を算出する。

ただし教育分野に関しては、学生関連の項目と教育業界就業者関連の項目のウェイトを50%ずつにするため、2段階目で「学生関連」「教育業界就業者」それぞれの加重平均を求め、3段階目にそれら2項目の加重平均を求める。

以下では、計算手順の詳細を示す。

### 1段階目

項目ごとの男女比を算出する。男性値と女性値を $\mathrm{Wom}_i, \mathrm{Men}_i\ (i = 1, \ldots, 47)$ とする。ここで $i$ は都道府県を表す。まず、比率をとり

$$
R_i = \frac{\mathrm{Wom}_i}{\mathrm{Men}_i}\ (i = 1, \ldots, 47)
$$

項目によっては $R_i = \frac{\mathrm{Men}_i}{\mathrm{Wom}_i}$ とする。この比率が1である場合、平等である。逆に1から離れるほど、不平等となる。項目によっては、一部の都道府県で1以上の値をとるものがある。この1以上の値は、1という平等状態よりも良い状態を表すとはいえないため1以上の値を1に変換する。

$$
\tilde{R}_i = \min(1, R_i)\ (i = 1, \ldots, 47)
$$

この変換した男女比 $\tilde{R}_i$ を項目ごとの男女比とする。

2段階目の加重平均で標準偏差を使用するため、平均、分散、標準偏差を

$$
\bar{R} = \frac{\sum_{i=1}^{47} \tilde{R}_i}{47}, \quad s^2 = \frac{\sum_{i=1}^{47} (\tilde{R}_i - \bar{R})^2}{47}, \quad s = \sqrt{s^2}
$$

とする。この平均、分散、標準偏差の算出で世界経済フォーラムでは人口比によるウェイトを使用しているが、本指数では、都道府県の扱いを平等とするため、人口比は用いなかった。

### 2段階目・3段階目(教育分野のみ)

各分野ごとに男女比の加重平均を算出する。例として、行政分野の9項目の男女比を用いた加重平均を説明する。男女比 $\tilde{R}_i$ と標準偏差 $s$ に項目を表す $j\ (j = 1, \ldots, 9)$ を追加し、$\tilde{R}_{ji}, s_j\ (j = 1, \ldots, 9,\ i = 1, \ldots, 47)$ とする。ウェイトは、標準偏差の逆数から次の値

$$
w_j = \frac{\frac{1}{s_j}}{\sum_{j=1}^{9} \frac{1}{s_j}}\ (j = 1, \ldots, 9)
$$

を使用する。ウェイト $w_j$ は、項目ごとに一つの値が求まり、9項目であれば、9つのウェイトが算出される。行政分野の指数は、9項目の加重平均として以下の式で算出した。

$$
I_i = w_1 \tilde{R}_{1i} + w_2 \tilde{R}_{2i} + \cdots + w_9 \tilde{R}_{9i}\ (i = 1, \ldots, 47)
$$

なお、時系列での比較を単純化するために、ウェイトは2024年4月4日のデータをもとに算出した値で固定し、以後毎年同じ値を使用する。

## データ出典

2026年版で使用した30指標の出典と基準時点は以下の通り。基準時点は項目ごとに異なるため、各指標の括弧内に記載した。

### 政治分野

- 歴代知事の在職年数の男女比(2026年1月1日時点)
    - [全国知事会「知事ファイル」](http://www.nga.gr.jp/app/chijifile/)
        - 母数の年数として、沖縄は本土復帰後の年数、そのほかの都道府県は戦後からの年数を用いている
        - 男性の在職年数は、母数の年数から女性在職年数を引いて算出した

- 衆参両院の選挙区選出議員の男女比(2026年2月9日時点)
    - 共同通信選挙データベース
        - 参院の合区選出議員は、すべて自県選出議員として扱う

- 都道府県議会の男女比(2025年7月1日時点)
    - 全国都道府県議会議長会

- 女性ゼロ議会(2024年12月31日時点)
    - [総務省「地方公共団体の議会の議員及び長の所属党派別人員調」](https://www.soumu.go.jp/senkyo/senkyo_s/data/ninki/touhabetsu.html)
        - 女性ゼロ議会数については、女性ゼロの議会数を市町村ごとにカウントし、女性が在職している議会数を女性値とした。男性値は、男性が在職する議会は全議会なので、市町村の議会数を使用している
        - 本データは女性議員の人数ではなく、議会数であることに注意されたい

- 市区町村議会の男女比(2024年12月31日時点)
    - [総務省「地方公共団体の議会の議員及び長の所属党派別人員調」](https://www.soumu.go.jp/senkyo/senkyo_s/data/ninki/touhabetsu.html)

- 市区町村長の男女比(2026年1月1日時点)
    - 全国市長会、全国町村会のリストを基に算出
